**Tools to quick run experiment on PyTorch**

- exp.py: configure your experiments.
- collate.py: collate_fn.
- dist.py: distribution utils.
- sampler.py: data sampler.
- trainer.py: train model.
- utils.py: load model, ema, lr sche ...
- flops.py: get model params and flops.
- memonger.py: same as `torch.utils.checkpoints`.
